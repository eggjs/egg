## Installation

```bash
$ npm install egg --save
```

Node.js >= 8.0.0 required.

## Getting Started

Follow the commands listed below.

```bash
$ npm install egg-init -g
$ egg-init --type simple showcase && cd showcase
$ npm install
$ npm run dev
$ open http://localhost:7001
```
