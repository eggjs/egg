## Installation

```bash
$ npm install egg --save
```

Node.js >= 6.0.0 required.

## Getting Started

Follow the step

```bash
$ npm install egg-init -g
$ egg-init --type simple showcase && cd showcase
$ npm install
$ npm run dev
$ open http://localhost:7001
```
