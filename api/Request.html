queries :Array

获取当前请求以 querystring 传递的参数，所有参数值都以 Array 类型返回，类似 Request#query

Source:
  • app/extend/request.js, line 204
Example
```js
GET http://127.0.0.1:7001?a=b&a=c&o[foo]=bar&b[]=1&b[]=2&e=val
this.queries
=>
{
  "a": ["b", "c"],
  "o[foo]": ["bar"],
  "b[]": ["1", "2"],
  "e": ["val"]
}
```

query :Object

获取当前请求以 querystring 传递的参数，所有参数值都以 String 类型返回

Source:
  • app/extend/request.js, line 180
Example
```js
GET http://127.0.0.1:7001?name=Foo&age=20&age=21
this.query
=> { 'name': 'Foo', 'age': 20 }

GET http://127.0.0.1:7001?a=b&a=c&o[foo]=bar&b[]=1&b[]=2&e=val
this.query
=>
{
  "a": "b",
  "o[foo]": "bar",
  "b[]": "1",
  "e": "val"
}
```
