## 安装

```bash
$ npm install egg --save
```

支持 Node.js 6.x 以上版本。

## 特性

- ✔︎ 内置多进程管理
- ✔︎ 高度可扩展的插件机制
- ✔︎ 深度框架定制
- ✔︎ 丰富的[插件](https://eggjs.org/badgeboard/)

## 快速开始

```bash
$ npm install egg-init -g
$ egg-init --type simple showcase && cd showcase
$ npm install
$ npm run dev
$ open http://localhost:7001
```
