## 特性

- 内置多进程管理
- 高度可扩展的插件机制
- 深度框架定制
- 丰富的[插件](https://github.com/search?q=topic%3Aegg-plugin&type=Repositories)

> 支持 Node.js 8.x 及以上版本。

## 快速开始

```bash
$ mkdir showcase && cd showcase
$ npm init egg --type=simple
$ npm install
$ npm run dev
$ open http://localhost:7001
```
